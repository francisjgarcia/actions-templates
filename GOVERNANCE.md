# Governance

## Project maintainer

| Name | GitHub | Role |

## Decision making

As a single-maintainer project, decisions are made by the maintainer.
For significant changes, an issue will be opened for community discussion
before a decision is made.

## Contributions

All contributions are welcome via pull requests. See [CONTRIBUTING.md](CONTRIBUTING.md)
for the full process. The maintainer reviews and merges all PRs.

## Becoming a maintainer

If you are a regular contributor and are interested in becoming a maintainer,
please reach out at contacto@francisjgarcia.es.

## Changes to governance

Changes to this document must be proposed via pull request and approved
by the current maintainer.
